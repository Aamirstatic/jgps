# Jai Govind Public School Website

This is the official website for Jai Govind Public School, located in Joya, Amroha. The website is built using Next.js 14 and Tailwind CSS, providing a modern and responsive user experience.

## Features

- Responsive design that works on all devices
- Static site generation for fast loading
- SEO optimized with metadata and schema markup
- English language content
- Modern dark green (#035c00) color scheme
- Image gallery and slider
- Contact form

## Pages

- Home: Landing page with hero section, image slider, and key information
- About: Information about the school's history, mission, vision, and values
- Academics: Details about the curriculum and academic programs
- Admission: Admission process and requirements
- Facilities: Information about the school's infrastructure and facilities
- Gallery: Photo gallery showcasing school events and activities
- Contact: Contact information and form

## Development

### Prerequisites

- Node.js 18.x or later
- npm or yarn

### Installation

1. Clone the repository
2. Install dependencies:

```bash
npm install
# or
yarn install
```

### Running the Development Server

```bash
npm run dev
# or
yarn dev
```

### Building for Production

```bash
npm run build
# or
yarn build
```

### Exporting as Static HTML

```bash
npm run export
# or
yarn export
```
